Recent Projects

Predicting Credit Card Defaults Using Supervised Learning

A procedural notebook documenting the process for creating a supervised classifier. The highest performing model in terms of accuracy was a SVM, which achieved a .828 test accuracy.

Bayesian Regression Using Atmospheric CO2 Data

A look into just how powerful Bayes' rule can be. Given a function to model the concentration of CO2 in the atmosphere, relevant parameters are sampled using Markov chains and RSTAN. A comprehensive before and after study is completed with the final result capturing the CO2 levels quite accurately.

Image Classification Using Unsupervised Learning

This project dives into convolutional neural networks and how they can be used to classify images, specifically images of various cat breeds. There is a comparison between a CNN from scratch, the DenseNet pre-trained model and a DenseNet model with additional tuned layers.

Exploring Word Embeddings Using Natural Language Processing

A notebook which looks at multiple aspects of natural language processing. Generating word vectors, comparing word vectors by library, exploring bias in vectors and visualizations of word vectors using dimensionality reduction are all topics explored.
